How to setup the website

1. Please ensure that Visual Studio 2019 Express Edition is installed on the machine with local database setup.

2. The link to install Visual Studio 2019 Express Edition: Download Visual Studio 2019 Express Edition

3. Open SandysBakery.sln file with Visual Studio 2019.

4. The database for the application is located inside App_Data folder.

5. Once the solution file opens in the Visual Studio, click on IIS Express (Google Chrome) and the application will start building.

6. The application will launch in Google Chrome browser.

7. Use the credentials mentioned above to navigate through to the application.

Note: The application is supported only on Google Chrome, FireFox, Microsoft Edge and Opera.
